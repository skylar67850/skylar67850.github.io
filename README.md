# skylar67850.github.io

Source code for my personal website hosted with GitHub Pages.

## Overview

This repository contains the code for my website, which serves as a central place for projects, blog content, and other work I'm building and maintaining.

The site is built as a lightweight single-page application using standard web technologies with a custom terminal-inspired design.

## Features

* Responsive layout
* Single-file deployment
* Terminal-style interface
* Matrix-inspired visual effects
* Project showcase
* Blog integration
* Fast static hosting through GitHub Pages

## Tech Stack

* HTML5
* CSS3
* JavaScript
* GitHub Pages

## Development

Clone the repository:

```bash
git clone https://github.com/skylar67850/skylar67850.github.io.git
```

Run locally by opening `index.html` in a browser or serving the repository with any static web server.

## Deployment

The site is automatically deployed through GitHub Pages when changes are pushed to the default branch.
